# Быстрая загрузка на GitHub - 3 минуты

## Самый простой способ (без программ)

### 1. Создать репозиторий (30 секунд)
1. Откройте https://github.com/new
2. Repository name: `masha-clicker`
3. Public
4. Create repository

### 2. Загрузить файлы (1 минута)
1. Нажмите "uploading an existing file"
2. Перетащите ВСЕ файлы из папки:
   - index.html
   - style.css
   - script.js
   - telegram.js
   - fluid.js
   - theme-helper.js
   - 1.bmp, 2.jpg, 3.jpg, 4.jpg, 5.jpg, 6.png
   - README.md
3. Commit changes

### 3. Включить сайт (30 секунд)
1. Settings → Pages (слева)
2. Source: `main` → `/ (root)` → Save
3. Подождите 2 минуты

### 4. Получить ссылку (10 секунд)
1. Обновите страницу Settings → Pages
2. Скопируйте ссылку:
   ```
   https://ваш-username.github.io/masha-clicker/
   ```

## Готово! 🚀

Откройте ссылку в браузере - игра работает!

---

## Для Telegram бота

1. Откройте @BotFather
2. `/newbot` → создайте бота
3. `/newapp` → добавьте Web App
4. Вставьте вашу ссылку с GitHub Pages
5. Готово!

---

## Если нужно обновить файлы

1. Откройте файл на GitHub
2. Нажмите карандаш ✏️
3. Измените
4. Commit changes
5. Подождите 1 минуту - изменения применятся

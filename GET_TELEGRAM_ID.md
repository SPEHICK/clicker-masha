# Как узнать свой Telegram ID

## Способ 1: Через бота (самый простой)

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Нажмите "Start"
3. Бот пришлет ваш ID, например:
   ```
   Id: 123456789
   ```
4. Скопируйте это число

## Способ 2: Через @getmyid_bot

1. Откройте [@getmyid_bot](https://t.me/getmyid_bot)
2. Нажмите "Start"
3. Получите ваш ID

## Способ 3: Через консоль браузера

1. Откройте вашу игру в Telegram Web App
2. Нажмите F12 (открыть консоль)
3. В консоли введите:
   ```javascript
   Telegram.WebApp.initDataUnsafe.user.id
   ```
4. Скопируйте число

---

## Что делать дальше

1. Узнайте свой Telegram ID (например: `123456789`)
2. Откройте файл `telegram.js` на GitHub
3. Найдите строку:
   ```javascript
   const DEVELOPER_TELEGRAM_ID = 'YOUR_TELEGRAM_ID';
   ```
4. Замените на ваш ID:
   ```javascript
   const DEVELOPER_TELEGRAM_ID = '123456789';
   ```
5. Сохраните файл (Commit changes)

Готово! Теперь только ваш Telegram аккаунт будет иметь права разработчика.

---

## Проверка

1. Откройте игру через Telegram бота
2. Перейдите в "Профиль"
3. Если вы разработчик - увидите кнопку "🔄 Сбросить прогресс"
4. Другие пользователи эту кнопку не увидят

## Важно

- ID - это число, например: `123456789`
- Не путайте с username (@shakall1488)
- ID уникален и не меняется
- Храните ID в секрете для безопасности
